
					BUBBLE SORT:

Este algoritmo apresenta complexidade de tempo médio, e pior caso de O(n2), sendo mais lento para conjuntos de dados maiores que 100.000 elementos. No melhor caso, quando a lista já está ordenada, a complexidade do tempo do Bubble Sort é O(n).

Complexidade espacial: complexidade O(1), o que significa que o espaço necessário não aumenta com o tamanho da entrada. Isso ocorre porque o Bubble Sort é um algoritmo de ordenação in-place, o que significa que ele não requer espaço adicional significativo além do que já é usado para armazenar a entrada.
